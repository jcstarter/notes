Git Notes

  • Git is a version control system that runs on your computer to track changes
  • Projects, aka, Repos can be pushed to a server - services like GitHub and BitBucket
  • To start tracking changes locally, use cd /Users/JC/directory/folder - Then run git init
  • Add the files you want git to track with git add FILE for a single file or git add --all or git add -A for all the files in the folder
  • After git add - files are now staged - to commit, use git commit -m 'Commit message'
  • Run git status to check status after action

Git Related Notes

  • It is recommended to create a readme file - either as .txt or .md (markdown language)
  • To push the project to GitHub - do git remote add origin remote repository URL then git push -u origin master

Terminal Notes (Eventually to be moved to its own page)

  • To open a file with a program in Terminal, type program file.type
  • To instantly create files, type touch file.type
